# Udemy Clone – Frontend Design 🎓

This project is a **frontend clone of the Udemy website**, built to practice and showcase modern web design skills.  
It focuses on layout, styling, and responsiveness using core web technologies.

## 🚀 Features
- Udemy-inspired homepage design
- Courses section with cards
- Navigation bar and footer
- Clean and user-friendly UI
- Responsive layout (basic)

## 🛠️ Technologies Used
- HTML5  
- CSS3  


## 📂 Project Structure
Udemy-Clone/
│── index.html
│── style.css
│── images/

## 💡 Purpose of This Project
- Improve frontend development skills
- Practice real-world UI cloning
- Learn layout design and CSS styling
- Build a strong portfolio project
